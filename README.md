# 🌙 Dream Interpreter
An AI-powered dream analysis app that helps you understand the deeper meanings behind your dreams through compassionate interpretation and symbolic analysis.

## ✨ Features
- AI-Powered Interpretation - Uses Claude AI to analyze dreams and provide thoughtful, psychological insights
- Symbolic Analysis - Identifies key symbols in your dreams and explains their potential meanings
- Emotional Understanding - Captures the emotional atmosphere and themes present in your dreams
- Personal Insights - Offers personalized reflections on what your dream might mean for you
- Guided Reflection - Provides gentle guidance for self-reflection and personal growth
- Multi-language Support - Available in English and Spanish (easily extensible to more languages)
- Dream Journal Export - Download all your dream interpretations as a formatted text file
- Beautiful UI - Dreamy gradient interface with floating cloud animations

## 🚀 How It Works

1. Describe your dream in as much detail as you can remember
2. Click "Interpret Dream" to get an AI-powered analysis
3. Explore the main themes, emotional atmosphere, and symbolic meanings
4. Reflect on personal insights and guidance provided
5. Download your dream journal anytime to keep a record

## 🛠️ Built With
- React
- Tailwind CSS
- Anthropic Claude API
- Lucide React Icons

## 🎨 Design Philosophy
The app creates a calming, mystical atmosphere with a nighttime gradient color scheme and gentle floating cloud animations to put users in a reflective mindset for exploring their subconscious.
